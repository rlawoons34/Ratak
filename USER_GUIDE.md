# 🏓 TakuRating 사용자 가이드

> **"직감이 아닌 데이터로, 당신의 탁구를 증명하세요."**
> TakuRating은 대학 탁구 동아리를 위한 실시간 데이터 분석 및 랭킹 플랫폼입니다.

## 🚀 1분 요약: TakuRating으로 할 수 있는 것
1. **실시간 서열 확인**: 내 랭킹이 몇 위인지, 라이벌은 어디에 있는지 확인하세요.
2. **승부 예측**: "내가 저 사람을 이길 수 있을까?" AI가 승률을 알려줍니다.
3. **성장 기록**: 내 실력이 늘고 있는지 그래프로 한눈에 볼 수 있습니다.
4. **계정 불필요**: 별도 가입 없이, 관리자가 등록만 해주면 바로 랭킹에 들어갑니다.

## 1. 🏆 랭킹 시스템 (Dashboard)

**"지금 내 순위는 몇 위일까요?"**

경기가 끝나고 결과가 입력되는 **0.1초 만에 랭킹이 변동**됩니다.

### 🔍 똑똑하게 랭킹 보는 법 (필터 기능)

단순한 전체 1등뿐만 아니라, 내가 속한 그룹에서의 순위를 확인해보세요.

* **🏫 학교별**: 한양대, 연세대 등 우리 학교 순위만 보기
* **🎓 대학부별**: 0부 / 1부 / 2부 / 3부 (대학 연맹 기준)
* **🏅 동아리부별**: -2부 ~ 8부 (동아리 자체 등급, 숫자가 낮을수록 높은 등급)

**예시:**
```
"한양대 대학 1부 선수들 중에서 내 순위는?"
→ 학교: 한양대, 대학부: 1부 필터 적용

"동아리 0부(최상위) 전체 순위는?"
→ 동아리부: 0부 필터 적용
```

### 📊 랭킹에 표시되는 정보

- **현재 레이팅**: 예) 1,650점
- **총 경기 수**: 몇 경기를 뛰었는지
- **승률**: 예) 68.5%
- **최근 30일 변화**: +42점 ⬆️ 또는 -15점 ⬇️

## 2. ⚔️ 전적 비교 & 승률 예측 (Compare)

**"한 번도 붙어본 적 없는 상대, 데이터는 결과를 알고 있습니다."**

[Compare] 메뉴에서 두 선수를 선택하면 3가지 분석을 제공합니다.

### ① 상대 전적 (Head-to-Head)

둘이 만났을 때 누가 더 많이 이겼는지 보여줍니다.

```
홍길동 vs 김철수
━━━━━━━━━━━━━━━
총 대결: 12회
홍길동 승: 7회 (58.3%)
김철수 승: 5회 (41.7%)
```

### ② 삼각 비교 (Triangle Logic)

공통 상대를 분석하여 안 붙어본 상대와의 상성을 예측합니다.

```
예시:
홍길동 vs 박선수: 3승 2패 (60%)
김철수 vs 박선수: 1승 4패 (20%)
━━━━━━━━━━━━━━━━━━━━━
결론: 홍길동이 김철수보다 박선수를 상대로 높은 승률
→ 홍길동이 김철수를 이길 가능성 높음
```

**⚠️ 순환 관계 감지**

A > B > C > A 같은 순환 관계도 자동으로 감지합니다.

```
홍길동 → 김철수 (홍길동 승)
김철수 → 이영희 (김철수 승)
이영희 → 홍길동 (이영희 승)
→ 순환 구조! 실력이 비슷한 그룹입니다.
```

### ③ AI 승률 예측

USATT/Elo 알고리즘으로 **"승리 확률 64%"** 처럼 정확한 수치를 제공합니다.

```
홍길동 (1750) vs 김철수 (1650)
━━━━━━━━━━━━━━━━━━━━━
홍길동 승리 확률: 64.0%
김철수 승리 확률: 36.0%
```

## 3. 👤 내 프로필 & 히스토리

**"나의 모든 탁구 역사가 기록됩니다."**

이름을 클릭하면 개인 프로필 페이지로 이동합니다.

### 📈 레이팅 그래프

내 실력의 상승/하락 곡선을 보여줍니다.

- 시간에 따른 레이팅 추이
- 주요 경기 포인트 표시
- 최고/최저 레이팅 기록

### 📝 경기 기록

누구와 붙어서 몇 점이 오르고 내렸는지(+13, -8) 상세하게 나옵니다.

```
2026-02-06 15:30
━━━━━━━━━━━━━━━━━━
홍길동 (1750) vs 김철수 (1650)
결과: 3-1 승리
레이팅 변화: 1750 → 1763 (+13)
```

### 🏆 대회 배지

리그전 우승, 오픈 대회 4강 등 나의 커리어가 배지로 표시됩니다.

- 우승 🥇
- 준우승 🥈
- 4강 🏅
- 8강 ⭐

### 📊 상세 통계

- **현재 레이팅**: 1,750점
- **총 경기 수**: 48경기
- **승률**: 68.8% (33승 15패)
- **최근 30일 변화**: +42점 ⬆️
- **연속 승/패**: 5연승 🔥

## 4. 🏫 학교 랭킹

**"우리 학교는 몇 위일까?"**

학교별 평균 레이팅으로 학교 간 대결을 즐길 수 있습니다.

```
1위: 서울대학교 (평균 1,682점)
2위: 한양대학교 (평균 1,658점)
3위: 고려대학교 (평균 1,642점)
```

학교를 클릭하면 해당 학교 소속 선수들의 명단을 확인할 수 있습니다.

## 8. 🔐 계정 시스템

### 계정이 없어도 참여 가능!

**비회원 선수**
- 관리자가 등록하면 즉시 경기 참여 가능
- 랭킹에 표시되고 레이팅 적용
- 계정 연동 없이도 사용 가능

**회원 가입 후 (선택사항)**
- 자신의 상세 기록 조회
- 레이팅 그래프 확인
- 경기 히스토리 열람
- 프로필 수정

### 회원 가입 방법

1. **회원가입 버튼** 클릭
2. 이메일 & 비밀번호 입력
3. 이메일 인증
4. **관리자에게 계정 연동 요청**
5. 연동 완료 후 내 프로필 확인

## 9. 🧮 레이팅 시스템 이해하기

### USATT 레이팅이란?

미국 탁구 협회(USATT)의 공식 알고리즘으로 계산되는 실력 지표입니다.

**기본 원리:**
- **초기 레이팅**: 1500점에서 시작
- **점수 변동**: 레이팅 차이에 따라 8~50점
- **강자를 이기면**: 많은 점수 획득
- **약자에게 지면**: 많은 점수 손실

### 점수 변동 예시

```
경우 1: 비슷한 실력 (레이팅 차이 50점 이내)
승자: +10점 / 패자: -10점

경우 2: 강자가 이김 (레이팅 차이 200점)
승자: +5점 / 패자: -5점

경우 3: 약자가 이김 (레이팅 차이 200점)
승자: +45점! / 패자: -45점
```

### 언더독 보너스 🎁

레이팅이 낮은 선수가 높은 선수를 이기면 **추가 보너스 점수**를 받습니다!

## 10. ❓ 자주 묻는 질문 (FAQ)

### Q. 랭킹에 제 이름을 올리고 싶어요.

**A.** 동아리 운영진(관리자)에게 **이름, 소속 학교, 부수**를 알려주세요.

- 대학부: 0부/1부/2부/3부
- 동아리부: -2부 ~ 8부 (0부가 최상위)

### Q. 경기 결과는 제가 직접 입력하나요?

**A.** 아니요, **공정성을 위해 '관리자'만 입력 가능**합니다.

경기가 끝나면 관리자에게 결과를 알려주세요:
- 승자/패자
- 점수 (예: 3-1)
- 경기 일시

### Q. 레이팅이 떨어질까봐 경기가 두렵습니다.

**A.** 레이팅은 실력의 정확한 반영입니다!

- 떨어지는 것도 성장의 과정
- 많이 경기할수록 정확해짐
- 약자를 이기면 적게, 강자를 이기면 많이 오름
- 장기적으로 진짜 실력에 수렴

### Q. 내 프로필을 볼 수 없어요.

**A.** 두 가지 경우가 있습니다:

1. **비회원 선수**: 랭킹에는 표시되지만, 로그인 없이는 상세 기록 볼 수 없음
2. **회원**: 회원가입 후 관리자에게 계정 연동 요청

### Q. 레이팅 계산이 잘못된 것 같아요.

**A.** 레이팅은 USATT 공식 알고리즘으로 자동 계산됩니다.

- 모든 계산이 투명하게 기록됨
- 경기 히스토리에서 각 경기의 점수 변화 확인 가능
- 문제가 있다면 관리자에게 문의

### Q. 다른 학교 선수와 경기 결과도 입력되나요?

**A.** 네! 모든 경기가 레이팅에 반영됩니다.

- 교내 경기
- 학교 대항전
- 오픈 대회
- 친선전

### Q. 오래 경기를 안 하면 레이팅이 떨어지나요?

**A.** 아니요, **활동 감소로 인한 페널티는 없습니다**.

레이팅은 경기 결과로만 변동됩니다.

### Q. 랭킹 1위가 되려면 어떻게 해야 하나요?

**A.** 꾸준히 강한 상대를 이기세요!

- 비슷하거나 높은 레이팅의 선수를 이겨야 많이 오름
- 약한 선수만 이기면 적게 오름
- 연속 승리로 모멘텀 유지

## 11. 📱 사용 시나리오

### 시나리오 1: 처음 시작하는 신입

```
1. 관리자에게 선수 등록 요청
2. 첫 경기 후 레이팅 1500점으로 시작
3. 여러 경기를 통해 내 실력 파악
4. 원하면 회원가입 후 계정 연동
5. 내 성장 그래프 확인
```

### 시나리오 2: 라이벌과의 대결

```
1. Compare 메뉴로 라이벌 분석
2. 승률 예측 확인 (예: 45%)
3. 전략 세우기
4. 경기 진행
5. 결과에 따라 레이팅 변동 확인
6. 다음 대결 준비
```

### 시나리오 3: 승급 도전

```
현재: 동아리 B등급 (레이팅 1580)
목표: A등급 진입 (1650 이상)
━━━━━━━━━━━━━━━━━━━━━
전략:
1. A등급 하위권 선수들과 매치업
2. 승률 50% 유지하며 점수 쌓기
3. 3~4주 간 꾸준히 경기
4. 목표 달성!
```

4. **성장 추적**

- **버그 제보**: 관리자에게 문의
- **계정 연동**: 관리자에게 문의
