# 🏓 TakuRating 사용자 가이드 (간단 버전)

**"직감이 아닌 데이터로, 당신의 탁구를 증명하세요."**

TakuRating은 대학 탁구 동아리를 위한 실시간 데이터 분석 및 랭킹 플랫폼입니다. 주관적인 '부수' 논쟁을 끝내고, USATT(미국 탁구 협회) 공식 알고리즘을 통해 산출된 객관적인 레이팅으로 나의 진짜 실력을 확인해 보세요.

---

## 1. 🏆 실시간 랭킹 (Dashboard)

**"지금 내 순위는 몇 위일까요?"**

홈 화면에서는 모든 등록 선수의 실시간 서열을 한눈에 볼 수 있습니다. 경기가 끝나고 결과가 입력되는 즉시 랭킹이 변동됩니다.

### 주요 기능
- **다중 필터**: 학교별, 대학부별(1부/2부/3부), 동아리부별(A/B/C/D) 원하는 랭킹만 골라보기
- **실시간 반영**: 경기 결과 입력 즉시 레이팅 & 순위 자동 업데이트
- **상세 통계**: 레이팅 점수, 총 경기 수, 승률, 최근 30일 변화량

---

## 2. ⚔️ 전적 비교 & 승률 예측 (Compare)

**"한 번도 붙어본 적 없는 상대, 내가 이길 수 있을까?"**

이 앱의 **가장 강력한 기능(Killer Feature)**입니다. 단순히 승패만 기록하는 것이 아니라, 데이터를 기반으로 승부를 예측합니다.

### 세 가지 분석 방식

**① 직접 대결 전적 (H2H)**
- 과거 몇 번 이기고 졌는지, 상대 전적 역사를 깔끔하게 정리

**② 삼각 비교 (Triangle Logic)**
- 나와 상대방이 공통으로 상대했던 '제3의 선수'와의 전적 분석
- 간접적으로 실력을 비교하여, 한 번도 붙어본 적 없는 상대도 예측 가능
- 순환 관계(A > B > C > A) 자동 감지

**③ AI 승률 예측**
- Elo 레이팅 알고리즘을 통해 내가 이길 확률이 60%인지, 40%인지 구체적인 수치로 제공
- USATT 공식 알고리즘 기반 정확한 예측

---

## 3. 👤 선수 프로필 (Profile)

**"나의 성장 과정을 기록하세요."**

나의 모든 탁구 역사가 기록되는 개인 페이지입니다.

### 제공 정보
- **레이팅 그래프**: 내 실력이 어떻게 상승(또는 하락)하고 있는지 시간별 추이
- **최근 경기 기록**: 최근 나의 컨디션이 좋은지 나쁜지 직관적인 승/패 배지로 확인
- **상세 통계**: 현재 레이팅, 전체 승률, 총 경기 수, 최근 30일 변화량
- **경기 히스토리**: 모든 경기의 상대, 결과, 점수 변동 상세 기록

---

## 4. 🏆 토너먼트 시스템 (Tournaments)

**"대회 기록도 완벽하게 관리하세요."**

### 대회 유형
- **리그전 (League)**: 정기 리그
- **오픈 대회 (Open)**: 누구나 참가 가능
- **선수권 (Championship)**: 공식 대회

### 대회 기록 관리
- 대회별 성적 (우승, 준우승, 4강 등)
- 조별 예선 순위
- 개인 프로필에서 참가 대회 전체 기록 확인

---

## 5. ⚡️ 월간 인사이트 (Monthly Insights)

**"이달의 주인공은 누구일까요?"**

단순한 랭킹 외에, 데이터를 뜯어보며 발견한 흥미로운 이야기들을 소개합니다.

- **이달의 이변 (Biggest Upset)**: 낮은 레이팅의 선수가 고수를 꺾은 '가장 충격적인 경기'
- **이달의 상승세 (Top Riser)**: 한 달 동안 실력이 가장 많이 늘어난 선수
- **이달의 핫 매치**: 가장 접전이었던 경기, 관심도가 높았던 라이벌전

---

## 6. 🏫 학교 랭킹 (Schools)

**"우리 학교 탁구부는 전국 몇 위?"**

개인전뿐만 아니라 학교 대항전의 재미도 놓치지 않았습니다.

- 각 학교 소속 선수들의 **평균 레이팅**을 산출하여, 학교 간의 자존심 대결
- 학교를 클릭하면 해당 학교 소속 선수들의 명단을 따로 모아보기
- 학교별 상세 통계: 선수 수, 평균 레이팅, 최고/최저 레이팅

---

## 🔐 사용자 권한 시스템

### 계정 없이도 참여 가능!

**비회원 선수**
- 관리자가 등록하면 즉시 경기 참여 가능
- 랭킹에 표시되고 레이팅 적용
- 계정 연동 없이도 사용 가능

**일반 회원 (Player)**
- 회원가입 후 선수 프로필 연동
- 자신의 상세 기록 조회 가능
- 레이팅 그래프, 경기 히스토리 확인

**관리자 (Admin)**
- 경기 결과 입력 권한
- 선수/학교/토너먼트 관리
- 모든 데이터 수정 가능

---

## 📝 경기 결과 입력

**공정성을 위해 지정된 관리자(Admin)만이 경기 결과를 입력할 수 있습니다.**

경기가 끝나면 관리자에게 결과를 알려주세요. 입력 버튼을 누르는 순간:

1. ⚙️ 서버가 **USATT 차트**를 기반으로 점수 자동 계산
2. 📊 양 선수의 **레이팅** 즉시 업데이트
3. 🏆 모든 **랭킹** 실시간 재정렬
4. 📈 **레이팅 히스토리** 자동 기록

### USATT 레이팅 시스템

- **초기 레이팅**: 1500점에서 시작
- **점수 변동**: 레이팅 차이에 따라 자동 계산 (8~50점)
- **언더독 보너스**: 낮은 선수가 이기면 추가 점수
- **공식 알고리즘**: 미국 탁구 협회(USATT) 공식 테이블 사용

## 🎯 시작하기

### 1단계: 랭킹 구경하기
- 회원가입 불필요
- 홈 화면에서 전체 랭킹 확인

### 2단계: 경기 참여하기
- 관리자에게 선수 등록 요청
- 등록 즉시 경기 참여 가능

### 3단계: 내 기록 보기 (선택)
- 회원가입 후 선수 프로필 연동
- 상세 통계 & 그래프 확인

---

## 💡 핵심 차별점

### ✅ 객관적 실력 측정
주관적인 '부수' 논쟁 종료. USATT 공식 알고리즘으로 정확한 실력 평가.

### ✅ 유연한 참여 방식
계정 없이도 경기 참여 가능. 원하면 나중에 연동.

### ✅ 이중 부서 관리
대학부(1/2/3부) + 동아리부(A/B/C/D) 각각 독립 랭킹.

### ✅ AI 기반 예측
한 번도 안 붙어본 상대도 삼각 비교와 Elo 알고리즘으로 승률 예측.

### ✅ 실시간 업데이트
경기 입력 즉시 모든 랭킹 자동 반영.
